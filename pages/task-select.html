💡 Щоб виділити кілька окремих елементів — тримайте Ctrl і клацайте по кожному потрібному.
  1. Клацніть по першому потрібному елементу.
  2. Натисніть і тримайте клавішу Ctrl.
  3. Не відпускаючи Ctrl, клацайте по решті потрібних.
  4. Виділили зайвий? Клацніть по ньому ще раз із Ctrl — виділення зніметься.

💡 Щоб виділити багато файлів поспіль (діапазон) — клацніть по першому, прокрутіть список і клацніть по останньому з Shift.
  1. Клацніть по першому файлу діапазону.
  2. Прокрутіть список донизу до останнього.
  3. Тримайте Shift і клацніть по останньому — виділиться все між ними.
Бонус (другий діапазон): Ctrl+клік по його початку, потім Ctrl+Shift+клік по кінцю. Саме Ctrl+клік переносить точку відліку на новий діапазон.

💡 Щоб виділити геть усі файли — натисніть Ctrl+A (від англ. All — «усе»).
  1. Клацніть один раз у списку, щоб він став активним.
  2. Натисніть Ctrl+A — усі файли виділяться миттєво.
  3. Бонус: після Ctrl+A зніміть виділення з одного файлу кліком по ньому з Ctrl.
