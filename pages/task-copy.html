Сторінка 1 — джерело для копіювання

Як працювати:
  1. Прочитайте підказку, що саме треба скопіювати з кожного запису.
  2. Виділіть мишею потрібний фрагмент тексту.
  3. Натисніть Ctrl + C (на macOS — ⌘ Cmd + C), щоб скопіювати.
  4. Перейдіть на Сторінку 2 і вставте у поле з тим самим номером (Ctrl + V, на macOS — ⌘ Cmd + V).
